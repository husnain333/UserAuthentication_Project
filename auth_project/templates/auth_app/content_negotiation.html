How Content Negotiation Works

Django REST Framework uses content negotiation to determine:

  • Which renderer to use for the response
  • Which parser to use for the request

In this case, we're using TemplateHTMLRenderer to return HTML instead of JSON.

View Code:

class ContentNegotiation(APIView):
    permission_classes = [AllowAny]
    renderer_classes = [TemplateHTMLRenderer]
    
    def get(self, request):
        return Response({"message": "Hello from DRF!"}, 
                       template_name="content_negotiation.html")
